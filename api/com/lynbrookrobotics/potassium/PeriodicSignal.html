PeriodicSignal 

Companion object PeriodicSignal

abstract class PeriodicSignal[+T] extends AnyRef

Represents a signal that requires a fixed update interval

This type of signal is used in cases where history must be tracked, as it prevents collisions between multiple value requests and includes a dt value for use in calculations.

T

the type of values the signal will produce
