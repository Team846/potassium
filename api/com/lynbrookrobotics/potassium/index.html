    Represents a single robotic component, which translates signal data into action

    Components can be though of as a function of command => Unit, which is implemented in applySignal. The function is expected to take the latest command and send it to hardware interfaces. In addition, the applySignal method is the place to implement safeties, as it is the last layer of signal transformation.
